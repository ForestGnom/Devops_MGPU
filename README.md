## Описание проекта
Интерактивный дашборд для анализа датасета пациентов (Healthcare Dataset from Kaggle). Визуализация и анализ данных о пациентах, заболеваниях и стоимости лечения.

## Структура проекта
├── data/ # Датасет с данными пациентов
├── notebooks/ # для Jupyter ноутбуков
├── src/ # Исходный код
│ ├── loader.py # Загрузчик данных
│ └── app.py # Streamlit дашборд
├── requirements.txt # Зависимости
└── README.md # Описание проекта
## Функциональность
- Загрузка и обработка датасета (>55,000 записей)
- Интерактивные фильтры по полу и заболеваниям
- Визуализация распределения пациентов
- Анализ стоимости лечения
